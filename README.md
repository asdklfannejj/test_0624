# Excel Data Visualization Web App

엑셀 파일을 업로드하면 첫 번째 시트를 읽어서 표로 보여주는 단일 파일 웹앱입니다.

## 기능

- `.xls`, `.xlsx` 업로드
- SheetJS CDN으로 엑셀 읽기
- 첫 번째 시트 자동 파싱
- 날짜 기준 기간 필터
- X축 / Y축 컬럼 선택
- 자동 요약 생성
- 보고서 생성
- PDF 다운로드

## 파일 구조

- `index.html`
- `style.css`
- `script.js`

## 실행 방법

브라우저에서 `index.html`을 열면 바로 사용할 수 있습니다.
